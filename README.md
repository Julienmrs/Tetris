## Readme Tetris

 Le tetris a été fait avec l'aide des bibliothèques utilisées en cours, GLFW, Glew, random et réalisé sur  QtCreator dans un wsl (Windows Subsystem for Linux).
 Pour le lancer il est nécessaire de posséder les différentes bibliothèques utilisées (et leur version utilisées) :
 
 - GLFW 
 - GLEW
 - GLM
 - cstdlib 
 - ctime

Au lancement du programme le jeu se lance et les pièces tombes les unes après les autres. Chaque ligne complète fait disparaitre les blocs la contenant, le jeu continu jusqu'à ce qu'il n'y ait plus de place pour pouvoir faire apparaître une nouvelle pièce. Lorsque le jeu s'arrête normalement de cette façon le nombre de bloc effacés est affiché dans la console.
